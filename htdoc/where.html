The current release is htdig-3.1.6.tar.gz. Note that this is a gzipped tar file. You will need the GNU gunzip program, part of the gzip package, to extract the files in the archive. You can get the latest version of any of the GNU tools from ftp://ftp.gnu.org/pub/gnu/

All releases of ht://Dig are also now available through SourceForge. Access is available through HTTP and FTP.

There are also previous releases and context diffs available through http://www.htdig.org/files/ along with contributed binary releases available through http://www.htdig.org/files/binaries/ Additionally, access is available for ftp through ftp.htdig.org.

The latest documentation of ht://Dig is always available at http://www.htdig.org/
